All "arrays" are really managed lists that have side effects when their objects are added or removed. They have:

  • add(?object?) // If no parameters provided, one is added with the default constructor
  • remove(object)
  • getList() // returns a copy of the array (you remove and add all you want.)
  • default
  • _memberType // must be set at initialization.
  • _addCallback
  • _removeCallback

Layout is complicated:

  • Each element must negotiate with the elements above and below to determine size (eg long y-labels)
  • Absolute styles -- apply inverse of current (scalex,scaley) to the absolutely-sized things like text, linewidth, etc.
