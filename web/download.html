Subversion

The best way to get the most recent version is from the Subversion version tracking service hosted at SourceForge.

  • Browse the repository at http://svn.sourceforge.net/viewvc/svgkit/
  • Command line:To check out the trunk, svn co https://svn.sourceforge.net/svnroot/svgkit/trunk svgkit
  • Windows using TortoiseSVN: After installing, right click where you want to check it out, select "SVN Checkout..." and type the url https://svn.sourceforge.net/svnroot/svgkit/trunk
